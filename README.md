# Gestión Inmobiliaria - Android App

## Objetivo

El sistema tiene como objetivo principal la **informatización de la gestión de alquileres de propiedades inmuebles** que realiza una agencia inmobiliaria.

## Funcionalidades Principales

* **Autenticación de Usuarios:** `Login` y `Logout` para el acceso del personal autorizado.
* **Gestión de Perfil:** Visualización y edición de los datos del usuario activo, incluyendo la opción de **cambio de contraseña**.
* **Pantalla Principal (Home):** Vista de inicio enfocada en la **ubicación geográfica de la inmobiliaria**.
* **Consumo de API:** Conexión e integración con la API sugerida en la documentación de la cátedra para la persistencia y consulta de datos.
* **Control de Inventario de Inmuebles:**
    * Listado completo de las propiedades disponibles.
    * Alta y registro de nuevos inmuebles al catálogo.
    * Sistema de **habilitación/deshabilitación** según su disponibilidad actual.

---

## Tecnologías Utilizadas

* **Lenguaje de Programación:** Java
* **Entorno de Desarrollo (IDE):** Android Studio
* **Compatibilidad:** Android API 28 (Android 9.0 Pie) o superior.
